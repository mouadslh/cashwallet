# Cash Wallet Management System

Full-stack app to track income/expenses, visualize stats, and set budgets.

- Frontend: React (Vite), MUI, Recharts
- Backend: Node.js (Express), MongoDB (or in-memory), JWT

## Run

Backend:

```bash
cd backend
cp .env.example .env
npm install
npm run dev
```

Frontend:

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```
